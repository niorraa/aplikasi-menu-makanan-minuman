APLIKASI MENU MAKANAN & MINUMAN

Aplikasi ini merupakan perangkat lunak sederhana berbasis Python yang dirancang untuk membantu pengguna dalam memilih menu makanan dan minuman secara praktis. Program ini menampilkan daftar menu lengkap dengan harga, serta menyediakan antarmuka berbasis terminal yang mudah dipahami dan digunakan oleh siapa saja.

Fungsi utama dari program ini adalah memberikan pilihan menu secara manual atau acak, menghitung total harga pesanan, serta memudahkan pengguna dalam membuat kombinasi makanan dan minuman. Dengan fitur seperti pemilihan beberapa item, pengacakan menu, hingga pembuatan combo otomatis, aplikasi ini menjadi alat yang bermanfaat untuk simulasi pemesanan atau pengambilan keputusan sederhana terkait pilihan menu.
